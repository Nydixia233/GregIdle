# GregIdle

GTNH（GregTech: New Horizons）主题的网页放置/挂机游戏（Idle/Incremental）。纯前端、单机、静态托管。

> 非商业 fan project，不隶属于 GTNH 官方团队。如果你觉得好玩，请去玩真正的 [GTNH](https://www.gtnewhorizons.com/)。

## 这是什么

把 GTNH 从石器时代到 UXV 的长线科技推进——采集、蒸汽、电力、化学、副产物、高阶自动化——做成放置游戏。不是 1:1 复刻，而是"放置翻译"：在忠实于 GTNH 材料链与时代节奏的前提下，将数千条配方和数百台机器抽象为放置游戏可玩的规模。

数值锚定 GTNH 2.9.0-beta-1 / GT5-Unofficial 5.09.52.594。

## 技术栈

React 18 + TypeScript · Vite · Tailwind CSS · Zustand（persist）· framer-motion · break_infinity.js · Vitest
